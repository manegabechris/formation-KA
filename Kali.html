Balises de métadonnées - **``** : Définit le titre du document affiché dans l'onglet du navigateur. - **`<meta>`** : Fournit des métadonnées comme la description, les mots-clés ou l'encodage des caractères. - **`<link>`** : Lien vers des fichiers externes comme les feuilles de style CSS. - **`<style>`** : Contient du CSS interne. ### 3. Balises de texte et de mise en forme - **`<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`** : Titres de niveaux différents (h1 étant le plus important). - **`<p>`** : Paragraphe. - **`<strong>`** : Texte en gras, indiquant une importance forte. - **`<em>`** : Texte en italique, indiquant une emphase. - **`<br>`** : Saut de ligne. - **`<hr>`** : Ligne horizontale pour séparer des sections. ### 4. Balises pour les listes - **`<ul>`** : Liste non ordonnée (avec des puces). - **`<ol>`** : Liste ordonnée (avec des numéros). - **`<li>`** : Élément de liste. ### 5. Balises d'ancrage et d'images - **`<a>`** : Lien hypertexte qui permet de naviguer vers une autre page ou ressource. - **`<img>`** : Insère une image dans la page. ### 6. Balises pour les tableaux - **`<table>`** : Définit un tableau. - **`<tr>`** : Ligne dans un tableau. - **`<td>`** : Cellule dans une ligne de tableau. - **`<th>`** : Cellule d'en-tête dans un tableau. ### 7. Balises multimédia - **`<audio>`** : Intègre un fichier audio. - **`<video>`** : Intègre un fichier vidéo. - **`<source>`** : Spécifie plusieurs sources pour l'audio ou la vidéo. ### 8. Balises de formulaires - **`<form>`** : Définit un formulaire pour la saisie utilisateur. - **`<input>`** : Champ de saisie pour différents types d'informations (texte, mot de passe, case à cocher, etc.). - **`<textarea>`** : Zone de texte multiligne. - **`<button>`** : Bouton cliquable. ### 9. Balises sémantiques Ces balises aident à structurer le contenu et à améliorer l'accessibilité : - **`<header>`** : En-tête d'une section ou d'un document. - **`<nav>`** : Section contenant des liens de navigation. - **`<main>`** : Contenu principal du document. - **`<article>`** : Article autonome qui peut être distribué indépendamment (ex: blog). - **`<section>`** : Section thématique dans un document. - **`<footer>`** : Pied de page d'une section ou d'un document. # Une balise bloc est une balise dont le contenu sera affiché en commençant sur une nouvelle ligne dans le document. Par exemple <H1>, <P> et <CENTER> sont des balises blocs, <B> n'en est pas une (on dit parfois dans ce cas que la balise définit un élément ``en ligne''). ### Conclusion Ces balises constituent la base du langage HTML et sont essentielles pour créer des pages web structurées et accessibles.
